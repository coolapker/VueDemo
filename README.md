# JS

```
"type" : "module"   // package.json文件
```
- `"type"`默认是CommonJS语法
- `"type"`指定为 `"module"`后为 ES6语法 

CommonJS导入语法：
```
let vue = require("vue")  
```
ES6导入语法：
```
import vue from 'vue'  
```
在Node环境，默认语法下使用ES6语法导入会报错
